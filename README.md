### Build from source

*Tested on: Pop!_OS 22.04 LTS*

```
sudo apt install cmake libsdl2-dev libbullet-dev libglm-dev libopenal-dev libglew-dev pybind11-dev python3-dev python3-sdl2 python3-opengl python3-zmq python3-pygments
make build
make run
```
